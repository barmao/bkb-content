# bkb-content

Open content for **[bkb.sh](https://api.bkb.sh/blog)** — a blog of thoughts and
experiments in building software.

Posts live here (not in the blog's private code) so anyone can contribute one with a
pull request. The blog pulls `posts/` and `authors.json` from this repo at build time.

- **Want to write something?** → [CONTRIBUTING.md](CONTRIBUTING.md)
- **Posts** → [`posts/`](posts/)
- **Authors** → [`authors.json`](authors.json)

This is an experimental pipeline. It only holds words and public profile links —
nothing sensitive.
